###### Найти расположение файла
```linux
locate nifi.properties
```
###### Запустить свою WSL-дистрибуцию
```bash
# В PowerShell/CMD набери:
wsl -l -v
# Эта команда покажет список дистрибутивов.
# Чтобы войти в определенный
wsl -d Ubuntu-22.04
```
