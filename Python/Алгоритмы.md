## Сортировка (Пузырьковая)
```python
def sort_bubble(a):  
    for i in range(len(a)):  
        for j in range(0,len(a) - i -1):  
            if a[j] > a[j+1]:  
                a[j],a[j+1]=a[j+1],a[j]  
    return a  
  
print(sort_bubble([5,1,2,9,10]))
```
## Ряд Фибоначи
```python
from functools import lru_cache  

# @lru_cache это декоратор который позволяет хранить результат работы функции, что ускоряет расчеты типовых операций, руглировка истории хранеиня черех maxsize происходит 
@lru_cache(maxsize=None)
# :int и ->int это документирование на то какой тип данных ожидаетс на входе и какой тип данных вернетс яна выходе
def fib(n:int) -> int:
	if n < 2:
		return n
	return fib(n-1) + fib(n-2)
```
